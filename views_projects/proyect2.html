Sistema de Gestión de Inventario Hotelero

Aplicación de escritorio diseñada específicamente para la gestión de inventario en hoteles. Permite un control eficiente de suministros, equipamiento y recursos del hotel.

Funcionalidades Principales:

  • Control de stock
  • Gestión de proveedores
  • Alertas de reabastecimiento
  • Reportes estadísticos
  • Control de costos
